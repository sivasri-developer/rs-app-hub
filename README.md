# RS App Hub 🛍📺🍔

A smart web app that brings all your favourite shopping, food and OTT apps in one place.

## Features
- 🔐 Login system with session memory
- 🔍 Live search bar
- 🗂 Category filter (Shopping / Food / OTT)
- 📱 Mobile responsive
- 🚀 No backend needed — runs in any browser
